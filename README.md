## Algoritmo de Predicción de Consumo de Combustible en Camiones CAEX

Este repositorio contiene el trabajo realizado por nuestro equipo, en el Bootcamp de Análisis de Datos para la Minera 4.0, 
(auspiciado por BHP y Universidad Andrés Bello), 
que permite predecir el consumo de combustible en Camiones CAEX, entregandole informacion importante a la minera para lograr 
gestionar el orden de carga de combustible y lograr reducir paros no programados por falta de combustible.
Esto se traduce en un numero mayor de camiones preparados para poder continuar faena y 
directamente ayuda a incrementar el nivel de producción real

### Contiene:

- Código Fuente del Algoritmo
- Infografía sobre Camiones CAEX y su consumo
- Presentación que explica el funcionamiento del algoritmo y su aporte a la Minera.
